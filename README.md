# Tether Point Site

## Локальный запуск

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Загрузите папку/репозиторий на GitHub.
2. В Vercel нажмите Add New Project.
3. Выберите репозиторий.
4. Framework Preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

Telegram в коде: `https://t.me/TetherPointExchange`.
